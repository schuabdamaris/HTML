Qual a diferença entre Clients e Servers?

(voltar)

Primeiramente vamos entender qual é a difenrença de um Client e um Server.

Como estamos vendo na imagem, os Clientes, que são os Clients, são os dispositivos onde nós utilizamos como meio de pesquisa no dia a dia, ele sozinho não tem a capacidade de arquivar tudo que a internet nos fornece por isso temos os Servidores, que no caso são os Servers. Quando queremos alguma informação os nossos dispositivos como clientes pede essa informação para a internet onde transmite para os servidores que pegam essa informação no banco de dados que repassa para a internet para, assim, chegar até nós.

Mesmo parecendo muito trabalhoso, nós temos todas as informações apenas em um click.

Como é uma estrutura básica de um HTML?

(voltar)

Agora vamos ver um pouco sobre a estrutura de alguns sites que se utilizam o HTML como base.

Aqui já vemos que precisamos de umas tags bases que são:

  • O html: que é para o servidor já saber que é um site se montando.
  • O head: que é para se montar a parte "por tras das câmeras", que é o que será preparado antes de ser apresentado para um usuário.
  • O title: ele funciona para dar o nome para a aba do navegador.
  • O body: que é o complemento do head, aqui já é tudo que o usuário irá ver, tudo que será posto em um site.
  • O h: ele já é a parte para ser escrita, o h ele é o título da página, que pode ir do 1 ao 6, do maior título até o menor título.
  • O p: esse já é os parágrafos, onde as informações serão descritas.
  • A /: se você percebeu essa barrinha está sempre para a palavra duplicada, mas para que ela serve? Bem, ela serve para se fechar a tag, para o seu navegador perceber o limite que a tag tem.

Como vimos o head não funciona sem o title e o body sem o h e o p para eles funcionarem, se não apenas teremos uma página em branco.

Também temos umas tags muito interessantes para se trabalahar. São essas:

  • O strong: que é deixar a palavra que estiver com ela ficar em negrito.
  • O i: que é para deixar a plavra que estiver com ela em itálico.
  • O u: que é para deixar a palavra que estiver com ela com um sublinhado embaixo dela.

Entendendo a diferença entre listas ordenadas e não ordenadas

(voltar)

Agora vamos falar sobre como montar uma lista em HTML.

Temos essas duas opções de listas. As ordenadas (ol) e as não ordenadas (ul). Mas o que seria uma lista ordenada? É como o nome diz, ela tem uma ordem, são mais facilmente encontradas como 1, 2, 3, ..., e por ai vai. E as não ordenadas? Essas já são as que não precisam de uma ordem em si, são encontradas com marcadores como na imagem.
